Enrolling Users in E-Learning Courses

How Superadmins and Power Users enroll learners in single or multiple e-learning courses in Docebo

Overview

01

Users can be enrolled in courses by Superadmins or Power Users with granted permissions, or they can self-enroll. This guide describes how Superadmins and Power Users can enroll users in e-learning courses — individually or in bulk across multiple courses.

Single Course Enrollment

02
Vendor Resource: See the Docebo Help Article for additional reference.
  1. 1
    Navigate to the Courses PageIn the left sidebar, hover over the folder icon and select Content & Delivery › Courses to open the page displaying the list of courses.
  2. 2
    Select the CourseClick the title of the course to open the course configuration page.
  3. 3
    Enroll the User(s)In the top right corner, click the Manage enrollments button and select Enroll users.
  4. 4
    Set the Enrollment AttributesThe Additional information tab displays the Enrollment attributes section. Configure the enrollment priority — Mandatory, Required, Recommended, or Optional. The enrollment priority is displayed on the course card in the learner's My courses and learning plans page.

Multiple Course Enrollment

03
Vendor Resource: See the Docebo Help Article for additional reference.
  1. 1
    Navigate to the Courses PageIn the left sidebar, hover over the folder icon and select Content & Delivery › Courses to open the page displaying the list of courses.
  2. 2
    Select the CoursesSelect the checkboxes next to each of the courses you want to enroll users in.
  3. 3
    Open the Enrollment ActionIn the bottom right corner, click the Choose action button and select Enroll users.
  4. 4
    Enroll User(s)In the right panel that opens, go through the steps — Users, Additional information, and Notifications — to enroll users in the selected courses. Click Next after each step to proceed.
  5. 5
    Set the Enrollment AttributesThe Additional information tab displays the Enrollment attributes section. Configure the enrollment priority — Mandatory, Required, Recommended, or Optional.
Notice: A background job is started to process enrollments if the total number of enrollments exceeds 20. Remember that one enrollment is a combination of a single user and a single course — selecting 3 courses and 2 users creates 6 enrollments.
